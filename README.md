# DLB-minicool 项目

Dyson 风扇工作测试文档 / Dyson Fan Functional Test Specification

## 功能描述

### 关机状态
- 所有 LED 关闭，关机状态下按加键或者减键都可以查看电量
- 只要是关机状态，都要进入低功耗

### 开机
- 开机键拨到开机状态，等待大约 1 秒后才进入 1 档工作
- 在充电过程中开机工作也是一样
- 风速变化都是逐渐增加/减小的，不是瞬变的

### 风速调节
- 长按加键/减键 500ms 左右，风速会从一档逐渐加/减到松手时到达的档位
- 如果加到五档按键继续长按就会进入超频模式
- 如果已经减到 1 档后，就不能往下减了
- 加档或者减档的时间是 500ms 左右加一或者减一

### LED 工作状态（从左往右）
| 风速档位 | LED 状态 |
|---------|---------|
| 一档 | 第一颗 LED 亮 |
| 二档 | 第一、二颗 LED 亮 |
| 三档 | 第一、二、三颗 LED 亮 |
| 四档 | 第一、二、三、四颗 LED 亮 |
| 五档 | 第一、二、三、四、五颗 LED 亮 |
- 风速为一档时，每短按一次减键，第一颗 LED 闪两次
- 风速为五档时，每短按一次加键，第五颗 LED 闪两次
- 如果长按加键约 1 秒左右，进入超频模式，按键板所有 LED 开始进行呼吸灯，松手后结束呼吸灯，变回 5 档

### 低电量
- 低电量时，充电板红灯常亮，关机熄灭

### 关机充电
- 关机充电时，充电板白灯呼吸灯，按键板的 LED 会分电量来显示：
  - 电量 0%-20%：第一颗 LED 呼吸灯
  - 电量 20%-40%：第一颗 LED 常亮，第二颗 LED 呼吸灯
  - 电量 40%-60%：第一、二颗 LED 常亮，第三颗 LED 呼吸灯
  - 电量 60%-80%：第一、二、三颗 LED 常亮，第四颗 LED 呼吸灯
  - 电量 80%-100%：第一、二、三、四颗 LED 常亮，第五颗 LED 呼吸灯
- 电量充满足之后，充电板白灯常亮，按键板的五颗 LED 常亮
- 充电过程中开机工作，按键板上正在工作的 LED 都会先全部熄灭，然后等待约 1 秒后 LED 固定在风速一档的状态

### 关机查看电量
- 关机模式下加减按键查看电量 LED 显示，五秒后熄灭，进入低功耗
  - 电量 0%-20%：第一颗 LED 亮，其它灭
  - 电量 20%-40%：第一、二颗 LED 亮，其它灭
  - 电量 40%-60%：第一、二、三颗 LED 亮，其它灭
  - 电量 60%-80%：第一、二、三、四颗 LED 亮，其它灭
  - 电量 80%-100%：第一、二、三、四、五颗 LED 亮

### 充电状态
- 关机正常充电时，LED 按照关机充电模式工作
- 如果在充电过程中开机工作，风速档位和 LED 显示都固定在一档
- 如果期间短按加键/减键，第一颗 LED 都会闪两下
- 如果再次关机，LED 则恢复关机正常充电工作模式

### 电量耗尽
- 所有板子的 LED 全部熄灭，重新开机没有任何反应
